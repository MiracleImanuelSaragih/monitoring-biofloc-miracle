🔗 Koneksi Sumber Data Sensor

Saat ini sensor belum mengirim data ke proyek Firebase ini. Untuk menghubungkan sensor (milikmu sendiri atau punya Firebase teman yang sudah berjalan), edit file config.js di project web ini:

  1. Cek di Firebase Console punya teman: Realtime Database atau Firestore? (lihat menu kiri).
  2. Minta teman menambahkanmu sebagai member project di Project Settings → Users and permissions.
  3. Salin firebaseConfig milik teman, ganti bagian FIREBASE_CONFIG di config.js.
  4. Cek struktur datanya di tab Realtime Database (pohon data), lalu sesuaikan bagian PATHS & FIELD_MAP di config.js supaya namanya cocok.
  5. Refresh web ini — data sensor akan otomatis muncul di Dashboard.

Lihat komentar lengkap & contoh di bagian atas file config.js.
