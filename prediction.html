🧠 Tentang Metode Prediksi

⬆️ Upgrade ke Random Forest (Opsional)

Saat data history sudah cukup banyak (disarankan 200+ baris, mencakup variasi kondisi normal & tidak normal), kamu bisa melatih model Random Forest Regressor di Python menggunakan data dari halaman History (export CSV). Setelah model dilatih, buat script Python kecil yang berjalan periodik (misal tiap 1 jam via cron/scheduler) untuk menghitung prediksi dan menulis hasilnya ke path Firebase: predictions/ph_forecast dengan format {predicted_ph, hours_ahead, generated_at}. Web ini otomatis akan memakai hasil Random Forest tersebut dan berhenti memakai metode regresi tren sederhana — tidak perlu mengubah kode web sama sekali.
